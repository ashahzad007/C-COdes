# Part 1 — Monday.com → HubSpot Sync Documentation

## Overview

This integration synchronizes **Monday.com board records** with **HubSpot CRM companies**.

Mapping rules:

| Monday Field | HubSpot Entity |
| ------------ | -------------- |
| Legal Name   | Parent Company |
| Site         | Child Company  |

If both exist → create **Parent–Child association** in HubSpot.

---

## Step 6 — Write Algorithm

```
FOR each Monday item

IF Site NULL AND LegalName NULL
    skip

IF Site NULL AND LegalName exists
    create/find parent

IF Site exists AND LegalName NULL
    create/find site

IF Site exists AND LegalName exists
    create/find parent
    create/find child
    create association
```

---

## Step 7 — Implement Code

Example pseudo code:

```csharp
foreach(var item in mondayItems)
{
    var site = item.Site;
    var legal = item.LegalName;

    if(site == null && legal == null)
        continue;

    if(site == null)
        CreateOrFindCompany(legal);

    else if(legal == null)
        CreateOrFindCompany(site);

    else
    {
        var parent = CreateOrFindCompany(legal);
        var child = CreateOrFindCompany(site);
        CreateAssociation(parent.Id, child.Id);
    }
}
```
